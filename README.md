### Getting Started
* Baixe o conteúdo, ou dê um forked no código.
* Instale e inicie o Expo-CLI 
``https://docs.expo.dev/``
* Tenha já pré configurado emulador Android ou IOS
* Instale o Expo no emulador que você utilizar
* Inicie o app
``expo start``
